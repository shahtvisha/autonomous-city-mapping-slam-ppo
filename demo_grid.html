What is this?
Occupancy grid built from real OpenStreetMap data — buildings are obstacles, roads are navigable space.

SLAM (Simultaneous Localisation and Mapping) builds the map in real-time from a limited sensor range — the agent can only see within its FOV circle.

PPO policy (Proximal Policy Optimisation) decides where to go next, trained to maximise road coverage and find electricity towers efficiently.

Frontiers (blue dots) = boundary between explored and unknown space — the key signal for exploration.
